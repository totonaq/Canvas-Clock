List of options

There are number of properties you can set for any given Clock instance.

Property Value Default Description
diam number 300 Set diameter to the clock (minimum 100px)
bgColor string '#fff' Set background color
borderWidth number 3 Set border width, px
borderColor string '#000' Set border color
centerDiam number 6 Set diameter to the closed nut, px
centerColor string #000 Set color to the closed nut
strokes boolean true Add or remove graduation to the clock
strokeMd boolean true Add or remove graduation (minutes only)
strokeSm boolean true Add or remove graduation (seconds only)
strokeLgWidth number 3 Set width to the large strokes (numbers 3, 6, 9 and 12), px
strokeMdWidth number 2 Set width to the medium strokes (hours), px
strokeSmWidth number 1 Set width to the small strokes (seconds), px
strokeLgLength number 20 Set length to the large strokes (numbers 3, 6, 9 and 12), px
strokeMdLength number 14 Set length to the medium strokes (hours), px
strokeSmLength number 8 Set length to the small strokes (seconds), px
strokeOffset number 40 Set offset between strokes and clock edge, px
strokeColor string '#000' Set strokes color
numerals false or string false Add or remove numerals (possible values: false, 'roman' or 'arabic')
numeralColor string #fff Set color to the numerals
numeralOffset number 20 Set offset between numerals center and clock edge, px
font string 'normal 16px Arial' Set font parameters to the numerals
arrowHWidth number 3 Set width to the hour hand
arrowHLength number 80 Set length to the hour hand
arrowMWidth number 2 Set width to the minute hand
arrowMLength number 100 Set length to the minute hand
arrowSWidth number 1 Set width to the second hand
arrowSLength number 120 Set length to the second hand
arrowColor string #000 Set color to the hands
enableSecArrow boolean true Show or hide second hand
tailLength number 15 Set length to the second hand tail
timeLabel boolean false Show or hide digital clock at the bottom
labelFont string 'Arial' Set font family to the time label
labelFontSize number 16 Set font size to the time label
labelFontWeight string '300' Set font weight to the time label
labelColor string '#000' Set font color to the time label
showDigitSec boolean false Show or hide digital clock seconds
timeZone false or string false Set UTC time zone offset (min UTC-12:00, max UTC+14:00). Possible values: 'current' (if you want to set your local time), '-3:00', '+5:30' etc. Possible values of minutes are 00, 30 and 45
dayLightSavingTime false or string false Set region of DST (false, 'NA' for North America, 'EU' for Europe, 'AUS' for Australia).
